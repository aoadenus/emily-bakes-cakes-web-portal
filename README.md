## Overview

This is a comprehensive business web application for a custom cake bakery, built as the final project for CIS 3343 at the University of Houston. The system covers the full customer journey — from browsing the menu and placing a custom order to staff processing and reporting.

Role-based dashboards give each team member (manager, sales, baker, decorator, accountant) a tailored view of the data relevant to their function. The project was presented on November 23, 2025, and reflects both technical execution and systems analysis thinking.

---

## Features

- 🌐 Customer-facing website with menu, about, contact, and order pages
- 🎂 Custom cake order form with real-time pricing, dynamic layer configuration (2–8 layers), and inline customer creation
- 💰 Preferred customer discount (10%) automatically applied at order time
- 👥 Five role-based staff dashboards: Manager, Sales, Baker, Decorator, and Accountant
- 📋 Order management with status tracking (pending, in progress, complete)
- 🗂️ Customer management with retail and corporate profiles, contact info, and order history
- 📊 Chart.js visualizations for revenue, order status, and top products on the staff dashboard
- 📱 Fully responsive design across all device sizes
